┌──────────────────────────────────────────────────────────────────────┐
│ ШАГ 1: ПОЛУЧИТЕ API КЛЮЧ OPENROUTER                                 │
└──────────────────────────────────────────────────────────────────────┘

1. Откройте в браузере: https://openrouter.ai/

2. Нажмите "Sign In" и войдите через Google/GitHub

3. Пополните баланс:
   → Перейдите: https://openrouter.ai/credits
   → Нажмите "Add Credits"
   → Пополните $5 (хватит на ~50,000 вопросов!)

4. Создайте API ключ:
   → Перейдите: https://openrouter.ai/keys
   → Нажмите "Create Key"
   → Дайте название: "CourseAutomation"
   → СКОПИРУЙТЕ ключ (начинается с sk-or-v1-...)

┌──────────────────────────────────────────────────────────────────────┐
│ ШАГ 2: НАСТРОЙТЕ CONFIG.JSON                                        │
└──────────────────────────────────────────────────────────────────────┘

1. Откройте файл: config.json

2. Найдите строку:
   "ai_api_key": "ВСТАВЬТЕ_ВАШ_КЛЮЧ_OPENROUTER_СЮДА",

3. Замените на ваш ключ:
   "ai_api_key": "sk-or-v1-ваш-настоящий-ключ",

4. Убедитесь что:
   "ai_enabled": true,

5. Сохраните файл (Ctrl+S)

┌──────────────────────────────────────────────────────────────────────┐
│ ШАГ 3: УСТАНОВИТЕ ЗАВИСИМОСТИ                                       │
└──────────────────────────────────────────────────────────────────────┘

Откройте командную строку в папке проекта и выполните:

   pip install -r requirements.txt

┌──────────────────────────────────────────────────────────────────────┐
│ ШАГ 4: ПРОВЕРЬТЕ РАБОТУ                                             │
└──────────────────────────────────────────────────────────────────────┘

Запустите тест:

   python test_ai.py

Должно быть:
   ✓ OpenRouter API инициализирован
   ✅ Все тесты пройдены успешно!

┌──────────────────────────────────────────────────────────────────────┐
│ ШАГ 5: ИСПОЛЬЗУЙТЕ!                                                 │
└──────────────────────────────────────────────────────────────────────┘

Запустите основной скрипт:

   python help.py

AI автоматически будет решать вопросы!
